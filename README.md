# Orbital Command Center

Orbital Command Center is a local TypeScript project that will grow into an agent command center inspired by the strategic feel of StarCraft. The long-term direction is a mission-control style workspace for coordinating agents, tracking operations, monitoring status, and running workflows entirely on a local machine.

This initial version is intentionally small. It sets up a clean TypeScript foundation, basic npm scripts, and a tiny local entrypoint so the repository can compile, run, and be published without committing to UI, orchestration logic, or themed assets yet.

## Current Scope

- Minimal TypeScript package structure
- Local development and build scripts
- Basic placeholder entrypoint
- Project description for the future command-center vision

## Planned Direction

In future iterations, Orbital Command Center may evolve toward:

- A local control surface for managing multiple agents
- Mission and task routing inspired by command-and-control interfaces
- Status dashboards, execution logs, and operator workflows
- A StarCraft-flavored visual identity without copying game assets directly

## Local Usage

```bash
npm install
npm run build
npm start
```
